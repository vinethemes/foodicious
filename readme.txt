== Description ==

Foodicious is a Free Food Blog WordPress Theme perfect for any blog or online food blogging website. This WordPress Personal Food Blog Theme features a three Sliders and Carousel and List, Simple and Grid Post styles which gives the website a clean and sturdy look.
The theme is best for personal food blogs and magazine websites. Theme Demo here https://www.vinethemes.com/themes?theme=Foodicious.

== Installation ==

1. In your admin panel, go to Appearance -> Themes.
2. Click Add New.
3. Click Upload.
4. Click Choose File and select the theme zip file you downloaded from WordPress.org.
5. Click Install Now.
6. Click Add New, then click Upload, then click Choose File.
7. After WordPress installs the theme, click Activate.
8. You've successfully installed your new theme!
